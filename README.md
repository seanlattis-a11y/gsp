# GSP Energy Site (static)

Energy-themed, corporate site for Geothermal Strategy Partners.

## Deploy on GitHub Pages
1. Create a new **public** repo (e.g., `gsp-website`).
2. Upload **all files at the repo root** (not inside a subfolder).
3. Settings → Pages → Source: **Deploy from a branch** → Branch: **main** / **/** (root) → Save.
4. Your site will publish at `https://<username>.github.io/<repo>/` within ~60 seconds.

## Customize
- Swap `assets/img/logo.svg` for your logo, or edit the `<img>` path in header.
- Colors live in `assets/css/main.css` under `:root{}`.
